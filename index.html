記憶相簿

網站簡介

這是一個簡潔美觀的照片分享網站，讓使用者可以輕鬆上傳、分類和展示自己的攝影作品或生活照片。無論是風景、人物、旅行日記，或是日常記錄，這裡都是屬於你的私人相冊，讓回憶隨時可見、輕鬆分享。

  • 目的：提供一個簡單、純粹的照片分享平台
  • 適用對象：喜歡攝影、生活紀錄的使用者

網站特色

  • 簡單介面：無廣告、無干擾的照片牆
  • 自由上傳：用戶可創建相簿上傳照片
  • 社群互動：留言、按讚功能
  • 主題標籤：方便搜尋同類型照片

網站主旨與設計理念

  • 打造一個結合 Instagram、Pinterest、Threads 元素的現代照片分享平台
  • 強調視覺吸引力與使用者互動性
  • 希望讓用戶能自由分享生活、瀏覽靈感、互相留言或按讚

核心功能規劃

  1. 照片上傳與編輯：支援濾鏡、裁剪、標籤功能
  2. 社交互動功能：留言、按讚、分享；追蹤其他用戶並建立個人動態牆
  3. 探索頁面設計：以圖片為主視覺呈現，支援主題標籤分類（如：旅行、美食、寵物）
  4. 個人頁面：顯示用戶簡介、照片牆、收藏清單
  5. 通知與私訊：即時互動提醒，簡易私訊系統

UI / UX 風格

  • 色彩乾淨、風格極簡
  • 使用卡片式佈局呈現圖片（類似 Pinterest）
  • 留白適中、重視手機端體驗

主題標籤

#旅行 #風景 #人物 #美食 #寵物 #日常
